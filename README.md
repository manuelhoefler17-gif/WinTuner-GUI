PACKAGE APPS AND UPLOAD
-----------------------------

After that you can search for the App you want to package.

<img width="1330" height="995" alt="image" src="https://github.com/user-attachments/assets/6e825da9-5760-4513-8ac3-1f5f6f73b3af" />

Click "Paket erstellen" to package it. After that you can klick "Upload to Tenant" and it is getting Uploaded to your tenant.



UPDATE APPS AND DELETE SUPERSEDED APPS
---------------------------------------

Click "Updates suchen" to search for apps with newer versions available. With "Ausgewählte App updaten" you upload the newest version of the selected app to the tenant (with supersed). 
If you click "Alle Apps updaten" then you update all apps.

<img width="1330" height="1097" alt="image" src="https://github.com/user-attachments/assets/c3be0e27-2d0b-4ddc-b0d1-e1c1bc8ad300" />

To delete the old app version you can click "Abgelöste Apps suchen".

Now you can select the app you want to delete and click "Ausgewählte App löschen".

<img width="1330" height="1097" alt="image" src="https://github.com/user-attachments/assets/b210309f-e0b3-49f6-a3fa-3cb4fcb0cdd2" />

If you want to delete all superseded apps you can click "Abgelöste Apps entfernen".

On both Options you getting ask if you really wanna delete this App versions.
